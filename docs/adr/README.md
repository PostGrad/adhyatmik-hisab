# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Adhyatmik Hisab PWA project.

## What is an ADR?

An ADR is a document that captures an important architectural decision made along with its context and consequences. We use the format proposed by Michael Nygard.

## Status Definitions

- **Proposed**: Under discussion
- **Accepted**: Decision made and implementation planned
- **Deprecated**: No longer relevant
- **Superseded**: Replaced by another ADR
